# Project-Civic
A track focused build log and design documentation project for an EK Honda Civic, built while studying Mechanical Engineering at Michigan State University.

## Goals
Design a Honda Civic into a race car on a limited budget with a primary focus on handling and endurance.

## Current Status
Pre-acquisition / Research Phase

## Structure
design-doc.md - Design rationale, decision log, and calculations
cad/ - CAD models and drawings
photos/ - build development photos
data/ - data log for measurements, corner weights, lap times
